=============================
Решение домашнего задания №1
=============================

Установка зависимостей
-------------------------------------------------------------------------------
- Установка Poetry, установка зависимостей из poetry.lock:

.. code::

    pip install poetry
    poetry install

Запуск приложения
-------------------------------------------------------------------------------
- Выполните в командной строке:

.. code::

    cd FastAPI\ base\ app/
    uvicorn app.main:app
